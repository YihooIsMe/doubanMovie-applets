<h2 align="center">模拟豆瓣电影微信小程序</h2>
<h3 style="margin:10px 0"> 小程序功能简介</h3>

>1、根据关键字搜索电影列表、下拉随机刷新关键字搜索.<br>
>2、电影详细详情页

<h3 style="margin:10px 0">下载运行</h3>

>1、[Download ZIP](https://github.com/YihooIsMe/doubanMovie-applets.git)
